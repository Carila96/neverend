## 修正3：振り子の当たり判定をsweep判定に変更

現在の2716行付近のコード：
```
if(Math.sqrt((pcx-tipX)*(pcx-tipX)+(pcy-tipY)*(pcy-tipY))<=16){lastDeathCause='spike';lastDeathStage=stageNum;lastTimeLeft=timeLeft;triggerDeath();return;}
```

これを以下に置き換えてください：
```
// Swept collision: check segment from prev frame to current frame against pendulum tip
const ax=player.prevX||pcx, ay=player.prevY||pcy;
const bx=pcx, by=pcy;
const cx=tipX, cy=tipY, r=16;
const dx=bx-ax, dy=by-ay;
const fx=ax-cx, fy=ay-cy;
const a=dx*dx+dy*dy;
const b=2*(fx*dx+fy*dy);
const c=fx*fx+fy*fy-r*r;
const disc=b*b-4*a*c;
let hit=false;
if(a<0.0001){hit=c<=0;}
else if(disc>=0){
  const t1=(-b-Math.sqrt(disc))/(2*a);
  const t2=(-b+Math.sqrt(disc))/(2*a);
  if((t1>=0&&t1<=1)||(t2>=0&&t2<=1)||(t1<0&&t2>1))hit=true;
}
if(hit){lastDeathCause='spike';lastDeathStage=stageNum;lastTimeLeft=timeLeft;triggerDeath();return;}
```
